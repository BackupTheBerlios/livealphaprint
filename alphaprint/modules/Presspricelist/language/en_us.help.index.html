Presspricelists

Use the Presspricelists module to create and manage pricelists for your organization. A presspricelist can be associated with multiple tasks and activities pertaining to one or more accounts and opportunities.

The Presspricelist Home page displays the following information:

  • A Search sub-panel where you can enter the presspricelist name to search for it. To find only pricelists assigned to you, select the Only my items box. To perform an advanced search using additional fields, click the Advanced Search tab. To customize and save the search layout and results, click the Saved Search & Layout tab.
  • A list of existing pricelists along with related information such as estimated time and actual effort.
    • To view the details of a presspricelist, click the name in the Presspricelist list.
    • To update or delete multiple pricelists, select the pricelists from the list and use the Mass Update sub-panel.
  • A Shortcuts section that displays the following options:
    • Create Presspricelist. Click this option to create a new presspricelist. Alternatively, you can use the New Presspricelist quick form to create the presspricelist and add enter additional information after you save it.
    • Presspricelist List. Click this option to navigate back to the Presspricelists Home page from a presspricelist's detail page.
    • Create Presspricelist Task. Click this option to create a new presspricelist task.
    • Presspricelist Tasks. Click this option to view a list of tasks for all your pricelists.
    • A New Presspricelist quick form that you can use to create a presspricelist with only the required fields. You can enter additional information after you save the presspricelist.
