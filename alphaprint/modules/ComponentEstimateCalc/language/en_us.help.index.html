ComponentEstimateCalcs

Use the ComponentEstimateCalcs module to create and manage componentestimatecalcs for your organization. A componentestimatecalc can be associated with multiple tasks and activities pertaining to one or more accounts and opportunities.

The ComponentEstimateCalc Home page displays the following information:

  • A Search sub-panel where you can enter the componentestimatecalc name to search for it. To find only componentestimatecalcs assigned to you, select the Only my items box. To perform an advanced search using additional fields, click the Advanced Search tab. To customize and save the search layout and results, click the Saved Search & Layout tab.
  • A list of existing componentestimatecalcs along with related information such as estimated time and actual effort.
    • To view the details of a componentestimatecalc, click the name in the ComponentEstimateCalc list.
    • To update or delete multiple componentestimatecalcs, select the componentestimatecalcs from the list and use the Mass Update sub-panel.
  • A Shortcuts section that displays the following options:
    • Create ComponentEstimateCalc. Click this option to create a new componentestimatecalc. Alternatively, you can use the New ComponentEstimateCalc quick form to create the componentestimatecalc and add enter additional information after you save it.
    • ComponentEstimateCalc List. Click this option to navigate back to the ComponentEstimateCalcs Home page from a componentestimatecalc's detail page.
    • Create ComponentEstimateCalc Task. Click this option to create a new componentestimatecalc task.
    • ComponentEstimateCalc Tasks. Click this option to view a list of tasks for all your componentestimatecalcs.
    • A New ComponentEstimateCalc quick form that you can use to create a componentestimatecalc with only the required fields. You can enter additional information after you save the componentestimatecalc.
