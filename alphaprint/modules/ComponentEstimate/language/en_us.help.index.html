ComponentEstimates

Use the ComponentEstimates module to create and manage componentestimates for your organization. A componentestimate can be associated with multiple tasks and activities pertaining to one or more accounts and opportunities.

The ComponentEstimate Home page displays the following information:

  • A Search sub-panel where you can enter the componentestimate name to search for it. To find only componentestimates assigned to you, select the Only my items box. To perform an advanced search using additional fields, click the Advanced Search tab. To customize and save the search layout and results, click the Saved Search & Layout tab.
  • A list of existing componentestimates along with related information such as estimated time and actual effort.
    • To view the details of a componentestimate, click the name in the ComponentEstimate list.
    • To update or delete multiple componentestimates, select the componentestimates from the list and use the Mass Update sub-panel.
  • A Shortcuts section that displays the following options:
    • Create ComponentEstimate. Click this option to create a new componentestimate. Alternatively, you can use the New ComponentEstimate quick form to create the componentestimate and add enter additional information after you save it.
    • ComponentEstimate List. Click this option to navigate back to the ComponentEstimates Home page from a componentestimate's detail page.
    • Create ComponentEstimate Task. Click this option to create a new componentestimate task.
    • ComponentEstimate Tasks. Click this option to view a list of tasks for all your componentestimates.
    • A New ComponentEstimate quick form that you can use to create a componentestimate with only the required fields. You can enter additional information after you save the componentestimate.
