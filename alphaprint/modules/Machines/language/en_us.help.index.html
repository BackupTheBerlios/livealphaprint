Machines

Use the Machines module to create and manage customer issues with your products.

The Machines Home page displays the following information:

  • A Search sub-panel where you can enter keywords such as the machine number and subject to search for specific machines. To restrain the search to machines related to items that are assigned to you, select the Only my items box. To perform an advanced search using additional fields, click the Advanced Search tab. To customize and save the search layout and results, click the Saved Search & Layout tab.
  • A list of existing machines along with related information such as bugs, activities, and history.
    • To view the details of a machine, click the name in the Machine list.
    • To export information on one or more machines in .csv format to your local machine, select them in the list, click the Export link, choose Selected Records, and save the file. To export all the machines listed on the page or the entire list, select Current Page and Entire List respectively.
    • To update or delete multiple machines, select them from the list, and use the Mass Update sub-panel.
  • A Shortcuts section that displays the following options:
    • Create Machine. Click this option to report a new machine.
    • Machines. Click this option to navigate back to the Machines home page from a machine's detail page.
  • A New Machine quick form that you can use to create a machine with only the required fields. You can enter additional details after you save the machine.
