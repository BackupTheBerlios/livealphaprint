ClientOrders

Use the ClientOrders module to create and manage clientorders for your organization. A clientorder can be associated with multiple tasks and activities pertaining to one or more accounts and opportunities.

The ClientOrder Home page displays the following information:

  • A Search sub-panel where you can enter the clientorder name to search for it. To find only clientorders assigned to you, select the Only my items box. To perform an advanced search using additional fields, click the Advanced Search tab. To customize and save the search layout and results, click the Saved Search & Layout tab.
  • A list of existing clientorders along with related information such as estimated time and actual effort.
    • To view the details of a clientorder, click the name in the ClientOrder list.
    • To update or delete multiple clientorders, select the clientorders from the list and use the Mass Update sub-panel.
  • A Shortcuts section that displays the following options:
    • Create ClientOrder. Click this option to create a new clientorder. Alternatively, you can use the New ClientOrder quick form to create the clientorder and add enter additional information after you save it.
    • ClientOrder List. Click this option to navigate back to the ClientOrders Home page from a clientorder's detail page.
    • Create ClientOrder Task. Click this option to create a new clientorder task.
    • ClientOrder Tasks. Click this option to view a list of tasks for all your clientorders.
    • A New ClientOrder quick form that you can use to create a clientorder with only the required fields. You can enter additional information after you save the clientorder.
