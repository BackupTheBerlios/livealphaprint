EstimateCalcs

Use the EstimateCalcs module to create and manage estimatecalcs for your organization. A estimatecalc can be associated with multiple tasks and activities pertaining to one or more accounts and opportunities.

The EstimateCalc Home page displays the following information:

  • A Search sub-panel where you can enter the estimatecalc name to search for it. To find only estimatecalcs assigned to you, select the Only my items box. To perform an advanced search using additional fields, click the Advanced Search tab. To customize and save the search layout and results, click the Saved Search & Layout tab.
  • A list of existing estimatecalcs along with related information such as estimated time and actual effort.
    • To view the details of a estimatecalc, click the name in the EstimateCalc list.
    • To update or delete multiple estimatecalcs, select the estimatecalcs from the list and use the Mass Update sub-panel.
  • A Shortcuts section that displays the following options:
    • Create EstimateCalc. Click this option to create a new estimatecalc. Alternatively, you can use the New EstimateCalc quick form to create the estimatecalc and add enter additional information after you save it.
    • EstimateCalc List. Click this option to navigate back to the EstimateCalcs Home page from a estimatecalc's detail page.
    • Create EstimateCalc Task. Click this option to create a new estimatecalc task.
    • EstimateCalc Tasks. Click this option to view a list of tasks for all your estimatecalcs.
    • A New EstimateCalc quick form that you can use to create a estimatecalc with only the required fields. You can enter additional information after you save the estimatecalc.
