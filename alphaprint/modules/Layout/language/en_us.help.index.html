Layouts

Use the Layouts module to create and manage layouts for your organization. A layout can be associated with multiple tasks and activities pertaining to one or more accounts and opportunities.

The Layout Home page displays the following information:

  • A Search sub-panel where you can enter the layout name to search for it. To find only layouts assigned to you, select the Only my items box. To perform an advanced search using additional fields, click the Advanced Search tab. To customize and save the search layout and results, click the Saved Search & Layout tab.
  • A list of existing layouts along with related information such as estimated time and actual effort.
    • To view the details of a layout, click the name in the Layout list.
    • To update or delete multiple layouts, select the layouts from the list and use the Mass Update sub-panel.
  • A Shortcuts section that displays the following options:
    • Create Layout. Click this option to create a new layout. Alternatively, you can use the New Layout quick form to create the layout and add enter additional information after you save it.
    • Layout List. Click this option to navigate back to the Layouts Home page from a layout's detail page.
    • Create Layout Task. Click this option to create a new layout task.
    • Layout Tasks. Click this option to view a list of tasks for all your layouts.
    • A New Layout quick form that you can use to create a layout with only the required fields. You can enter additional information after you save the layout.
