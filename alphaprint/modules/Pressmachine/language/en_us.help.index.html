Pressmachines

Use the Pressmachines module to create and manage pressmachines for your organization. A pressmachine can be associated with multiple tasks and activities pertaining to one or more accounts and opportunities.

The Pressmachine Home page displays the following information:

  • A Search sub-panel where you can enter the pressmachine name to search for it. To find only pressmachines assigned to you, select the Only my items box. To perform an advanced search using additional fields, click the Advanced Search tab. To customize and save the search layout and results, click the Saved Search & Layout tab.
  • A list of existing pressmachines along with related information such as estimated time and actual effort.
    • To view the details of a pressmachine, click the name in the Pressmachine list.
    • To update or delete multiple pressmachines, select the pressmachines from the list and use the Mass Update sub-panel.
  • A Shortcuts section that displays the following options:
    • Create Pressmachine. Click this option to create a new pressmachine. Alternatively, you can use the New Pressmachine quick form to create the pressmachine and add enter additional information after you save it.
    • Pressmachine List. Click this option to navigate back to the Pressmachines Home page from a pressmachine's detail page.
    • Create Pressmachine Task. Click this option to create a new pressmachine task.
    • Pressmachine Tasks. Click this option to view a list of tasks for all your pressmachines.
    • A New Pressmachine quick form that you can use to create a pressmachine with only the required fields. You can enter additional information after you save the pressmachine.
